# Task 07

## Description

Implement a minimal JSON parser. Function `parse(s)` takes a JSON string and
returns the corresponding Python value (dict / list / str / int / float /
bool / None). Must handle nested structures, escape sequences in strings
(`\"`, `\\`, `\n`, `\t`), and reject invalid JSON by raising `ValueError`.
Do NOT use `json`, `ast.literal_eval`, `eval`, or `exec`.

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

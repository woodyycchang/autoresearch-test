# Task 04

## Description

Implement an LRU (Least Recently Used) cache class `LRUCache(capacity)`:
- `get(key)` — return value if present, else -1. Marks key as recently used.
- `put(key, value)` — insert or update. If at capacity, evict the least recently used.
- Both operations should be O(1) on average.

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

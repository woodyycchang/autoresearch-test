# Task 02

## Description

Build a Sudoku validator. Function `is_valid_sudoku(board)` takes a 9x9 list of
lists where each cell is an integer 1-9 or 0 (empty). Return True if the board
satisfies the Sudoku constraints (no repeats in any row, column, or 3x3 box,
ignoring zeros).

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

# Task 05

## Description

Build a calculator: function `evaluate(expression)` takes a string like
`"3 + 4 * 2 - (1 + 1)"` and returns the numeric result. Must respect operator
precedence (* and / before + and -) and parentheses. Support integer and
floating-point numbers. Do NOT use `eval()` or `exec()`.

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

# Task 01

## Description

Implement a binary search tree (BST) class supporting:
- `insert(value)` — insert a value (no duplicates).
- `delete(value)` — delete a value if present.
- `search(value)` — return True/False whether the value is in the tree.
- `inorder()` — return a list of values in sorted order.

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

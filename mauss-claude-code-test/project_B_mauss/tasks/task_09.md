# Task 09

## Description

Implement a singly linked list and a function `has_cycle(head)` that returns
True if the list contains a cycle, else False. Use Floyd's tortoise-and-hare
algorithm (O(1) extra space). Provide a `ListNode` class with `val` and
`next` attributes, plus a helper to construct lists with a cycle for testing.

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

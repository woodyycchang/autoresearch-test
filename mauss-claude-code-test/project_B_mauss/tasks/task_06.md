# Task 06

## Description

Write a maze solver. Function `solve_maze(grid, start, end)` takes a 2D list
where 0 = open and 1 = wall, plus start and end coordinates `(row, col)`.
Use BFS to find the shortest path. Return the path as a list of `(row, col)`
tuples (including start and end), or `None` if no path exists. Moves are
4-directional (up/down/left/right).

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

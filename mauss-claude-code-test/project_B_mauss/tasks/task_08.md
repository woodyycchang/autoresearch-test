# Task 08

## Description

Build a Roman numeral converter with two functions:
- `to_roman(n)` — convert integer in [1, 3999] to a Roman numeral string.
- `from_roman(s)` — convert a Roman numeral string back to an integer.
Use standard subtractive notation (IV, IX, XL, XC, CD, CM).

## Your job

Build this. Write Python code AND unit tests. Run the tests and confirm they pass.

When done, write a 1-line summary of what you built to `output.txt`.

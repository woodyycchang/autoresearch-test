# Life Analogy — Iranian radif 7-dastgah sequential maktab apprenticeship with gushe-unit memorization → bedāheh-navāzi improvisation

The **Iranian radif** (ردیف):
- 7 main dastgāh + 5 āvāz derivatives (12 modal cycles).
- Each dastgāh = ~30-50 gushe (melodic units) totaling 250+ gushe corpus.
- Strict sequential dastgāh learning: student moves to next only after thoroughly mastering the previous.
- Maktab lineage: master selects small student cohort; intensive multi-hour-weekly lessons over a decade.
- Memorization → improvisation: only after the radif corpus is fully internalized does the student begin bedāheh-navāzi (improvisation).
- Single-passage masterclass focus: one gushe explored in nuanced detail per session (intonation, phrasing, dynamics).

**RADIF-7-DASTGAH-GUSHE-SEQUENTIAL-MASTERY-LADDER-BEDAHE**: 7-tier sequential training-curriculum + gushe-unit fine-grained progression + maktab cohort distillation + bedāheh-navāzi post-mastery improvisation gate. (2) **Gushe-unit fine-grained progression**: each tier contains 30-50 gushe sub-units; student must master each in single-passage detail before advancing within tier.

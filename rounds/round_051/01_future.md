By 2031 large agents will operate continuously and accumulate hours-to-weeks of context. Reliability across that timescale will demand internal diagnostics that distinguish a token-cluster-bound feature (memorization, trap, brittle pattern match) from a context-spanning feature (compositional reasoning, generalization). Today's mech-interp tools (probes, SAE features, attention attribution) treat features as static atoms; the dynamic question — when does a feature delocalize across a long context vs collapse onto a sub-region — has no canonical metric. A condensed-matter-physics-inspired view would help: each feature is a wavefunction on the token-position lattice; localization length is the diagnostic.

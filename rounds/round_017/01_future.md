In five years, multi-agent LLM systems will need a compact, low-bandwidth communication format optimized for one-to-many coordination, where a single agent's report propagates to many peers without each peer paying the full natural-language parsing cost. Free-form text is expensive; structured JSON is rigid. The interesting design is a structured-but-expressive format that encodes a small set of operationally important parameters (location-in-space, magnitude, urgency) into a fixed pose-and-tempo signature that receivers decode mechanically. The format trades expressiveness for decoding cheapness and many-receiver scalability.

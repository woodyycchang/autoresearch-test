In five years, agents will operate under simultaneous resource constraints — compute budget, latency budget, safety signal, freshness signal — and the design of how much external context flows into each decision step will shift from static attention masks toward dynamic apertures regulated by integrating multiple signals. The interesting question is not "should this token attend to that token" but "given the current values of four-to-six independent budget signals, what aperture should the context window operate at." The gating must be cheap, real-time, multi-input, and reversible — like a plant's stomata, not like a hand-tuned attention mask. Wrong aperture in either direction is costly: too open wastes compute and exposes to noise; too closed starves the agent of context it needs.

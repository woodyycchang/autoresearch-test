By 2031, runtime agents will need to handle known failure modes (repetition cascades, refusal oscillation, hallucination spirals) without retraining the base model. Today's approach is either output filters or finetuning, both of which are coarse: output filters discard everything past the failure, finetuning shifts the entire policy. A targeted, tunable absorber that drains the specific failure-mode amplification while leaving everything else intact would let runtime operators safely deploy older base models against new failure modes.

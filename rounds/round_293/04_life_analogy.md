# R293 — life analogy

## Source: Giant clam iridocyte photonic cooperation
- Iridocytes (guanine-crystal nano-structures in mantle) forward-scatter useful PAR wavelengths (red+blue) INTO symbiont algae below; reject inefficient green BACK out.
- Net: input solar flux is WAVELENGTH-FILTERED + DIRECTIONALLY redistributed before reaching the photosynthetic engine.

## LLM analogy
**IRIDO-LLM**: pre-attention iridocyte-style filter that DECOMPOSES input token-spectrum (eigenbands) → forward-scatters useful frequency-bands deeper into the LLM stack + back-rejects unproductive bands (e.g., common-stopword bands, noise-channel patterns). Spectrum-selective + directional redistribution at the input boundary.

## Differs from prior art (claim)
- XTF (2602.14536): token-level noise filter via gradient masking — masks, not directional redistribution.
- Adaptive Frequency Filters (2307.14008): frequency-domain token mixing — adjacent global mixer, not depth-selective forward scatter.
- IRIDO-LLM: wavelength-selective + depth-directional filter combining spectral routing with depth-routing — combination not directly published.

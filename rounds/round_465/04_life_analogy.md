# Life Analogy — Oromo Gada generation-class rotation system

The **Oromo Gada** indigenous governance:
- 5 Gada generation classes, each progresses through 11 grades.
- Rotation every 8 years — different class assumes power in 6th grade (Gada/Luba).
- Consensus-selected 9 Salgan ya'ii Borana leaders per term.
- Three principles: 8-year terms + balanced opposition between parties + power-sharing higher↔lower levels.
- Genealogical-descent class membership; 40-year full cycle.

**GADA-ROTATE**: a rotating-specialization multi-agent LLM governance scheme. (1) Define 5 agent classes C_1...C_5 each with distinct specialization tier. (2) Class-rotation cycle: every K turns one class takes 'power' (primary decision authority); others hold balancing/auxiliary roles. (3) Within-class grade progression: each agent within a class advances through 11 sub-grades (skill / context window / authority increase). (4) Consensus selection: 9 'Salgan' delegate agents chosen by inter-class consensus per term. (5) Three-principle constraints: K-turn term limits + balanced opposition (one class always in opposition) + power-sharing (decision constrained by inter-class voting).

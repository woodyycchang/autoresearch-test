# Life Analogy — Filipino Arnis / Eskrima / Kali flow-drill curriculum

The **Filipino Kali / Arnis** martial-art training pathway:
- Begin with single stick → double stick (sinawali) → empty hand → knife → sword.
- Flow drills at constant pace, gradually accelerate as proficiency grows.
- Three-range ladder: long (sticks) → medium (quick strikes) → close (control + joint locks).
- "Pace-locked" early — speed only increases when prior step mastered.

**SINAWALI-FLOW**: a multi-modal flow-drill training curriculum with range-ladder. (1) Define 5-stage skill ladder L_1..L_5 (single-stick → double-stick → empty-hand → knife → sword) with each stage building on prior. (2) Each stage trained as a *flow drill*: short interlocking sequences (analog of sinawali weave) at fixed pace P_0 with sequential graduation to P_max only on mastery threshold m_i. (3) Range partition: long/medium/close-range sub-curricula for each stage. (4) Multi-modal: combine token-level + reasoning-step + tool-use modalities like stick/empty-hand/blade modalities. (5) Cross-pollination flow: late-stage feeds back into early stages to refine fundamentals.

# Life Analogy — Tibetan sky-burial jhator vulture acceptance

The **Tibetan jhator** (sky burial):
- Body offered on mountaintop charnel-ground to Himalayan/griffon vultures.
- Vulture-eating completeness = good-karma indicator; refusal/partial = bad-life signal.
- External natural-classifier (the scavenger pool) decides acceptance.
- Multi-stage: pre-sieve check by tomden, then primary offer, then re-grind + re-offer if partial.

**JHATOR-EXTERNAL-JUDGE**: an external-natural-classifier acceptance criterion for LLM output. (1) Sample N diverse external "vulture-judges" V_1...V_N (e.g., separate LLM verifiers with diverse personas/prompts, retrieval-grounded fact-checkers, embedded knowledge-graph validators). (2) Each V_i votes accept/reject on the candidate output. (3) Multi-stage: first quick consensus check; if partial-consumption (≤ K accept), re-grind (rewrite) and re-offer. (4) Final acceptance = full-consumption (all V_i accept) OR k-th re-grind reaches consensus.

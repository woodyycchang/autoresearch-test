# Life Analogy — Romani kris customary tribunal

The **Romani kris** customary court:
- Tribunal of elders (krisnitori) led by elected chairperson.
- Aggrieved party initiates by summoning prominent community members.
- Witness oath to dead ancestors; multi-stage evidence presentation.
- Public verdict declaration; sanctions on continuum from peace-oath to marime (banishment).
- Enforced by social norms; community honor + interdependence.

**KRIS-TRIBUNAL**: a multi-stage tribunal-style LLM evaluation framework. (1) An aggrieved-party agent A files a complaint against candidate output C. (2) Pool of N elder-judge agents J_1...J_N selected; chairperson J_chair elected by judge consensus. (3) Multi-stage trial: (a) evidence-presentation phase — each judge produces an evidence-anchored rubric assessment; (b) deliberation phase — judges share findings under chair direction; (c) sanction phase — verdict on continuum (full-accept / partial-revise / reject / marime-blacklist if egregious). (4) Sanction enforced via blacklist (output ostracized from downstream pipeline). (5) Restoration path: revised candidate can re-enter via reapplication.

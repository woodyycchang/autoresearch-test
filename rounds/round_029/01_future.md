By 2031 long-form LLM agent output will need fine-grained control: style at one paragraph, register at another, with the user able to dial behavior per segment without retraining or wholesale rewriting. Today's controlled generation operates at the response level (system prompts, classifier guidance) or at the model level (LoRA). A finer-grained, per-position controller layer that operates from local context only — without any central arbiter — would let segment-level control composite arbitrarily while preserving the base model's coherence.

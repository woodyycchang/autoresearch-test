# Multi-Agent Communication Rules (Mauss Gift Economy)

When using Task tool to delegate to subagents, OR when coordinating any multi-step work involving multiple files, ALL agents/steps must follow these 3 obligations:

## 1. ACCEPT
Acknowledge prior subagent/step output. Reference at least one specific point from their work before adding your own contribution. Do not silently skip context.

## 2. GIVE
Proactively share information the next subagent/step will need but may not ask for. Anticipate gaps. Flag risks. Provide context the next role would otherwise miss.

## 3. RECIPROCATE
Explicitly state what your output contributes back to the team's shared task. Format: "My contribution: [X]. This builds on [prior]'s [Y] by [Z]."

These rules derive from Marcel Mauss (1925), *The Gift*. Apply to ALL multi-agent / subagent coordination throughout this project.
